Getting Help

One thing to note is that the exercise's CSS styling instructions might sometimes appear vague. This is intentional. If you run into a style that you don't know how to implement, or don't know the correct property, I want you to follow this process:

  1. Try it: Sometimes, there are style properties that I didn't show you or you haven't seen, but they can be obvious. For example, I might not show you that to change an element's font size, but if you try to use the property font-size, you might find out that it works!

  2. Look it up: If trying a few obvious property names doesn't work, seach for it online. This means looking things up on Google, visiting forum pages like Stack Overflow, or referring to CSS tutorials like W3Schools or MDN Reference. Once you find your answer, make sure to understand what each property is doing first, then implement it in your own code.

  3. Ask for help: If you can't understand the detailed instructions, or just want a bit more guidance on what certain properties do, it's time to look for a real person. Whether it's a classmate, friend, or instructor, sometimes it's necessary to ask your question to an experienced person if you can't find the proper resources online.

Coding is a lifelong learning process, so it's extremely important to learn how to learn. You won't always have an instructor beside you to give you all the answers, so it's a good idea to practice looking things up on your own before defaulting to asking for help.
